🛠️ Tech Stack

Built with open-source tools and free APIs

⚡

FastAPI

High-performance async Python backend with SSE streaming for real-time itinerary delivery

🦙

Ollama

Local LLM inference — run Llama3, Mistral or Gemma entirely on your own machine. No API costs.

🗺️

OpenStreetMap

Overpass API for live attraction and hotel data. Nominatim for geocoding any destination on earth.

🌤️

Open-Meteo

Real-time weather data with 5-day forecasts. No API key required. Used for weather agent.

🌍

Leaflet.js

Interactive maps rendered on destination pages with real geocoded coordinates.

📝

Marked.js

Real-time Markdown-to-HTML rendering for the streaming itinerary output.

🎨

HTML / CSS / JS

Pure vanilla frontend — no React or frameworks. Fast, lightweight, and easy to run anywhere.

📡

SSE (Server-Sent Events)

Real-time streaming protocol for live itinerary delivery — word by word as the AI generates it.

🚀 The Project

Designed and developed as a complete full-stack AI application

🚀

Built as a Mini Project — AI & Multi-Agent Systems

AI Trip Planner demonstrates the power of multi-agent AI applied to a real-world problem. Four specialized agents run in parallel, each with a distinct role — weather analysis, attraction curation, hotel recommendation, and itinerary synthesis — coordinated by a modern FastAPI backend running local LLMs via Ollama.

The frontend is built with pure HTML, CSS, and Vanilla JavaScript — no frameworks overhead. Maps are powered by Leaflet.js and OpenStreetMap, weather by Open-Meteo, and the local AI runs completely offline with zero ongoing API costs.
